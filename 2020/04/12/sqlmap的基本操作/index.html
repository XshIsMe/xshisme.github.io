sqlmap的基本操作

发表于 2020-04-12

这里我用sqli-labs来实验

输出等级

使用 -v <等级> 参数来设置输出等级，默认是1

0：只显示 Python 的 tracebacks 信息、错误信息 [ERROR] 和关键信息 [CRITICAL]；
1：同时显示普通信息 [INFO] 和警告信息 [WARNING]；
2：同时显示调试信息 [DEBUG]；
3：同时显示注入使用的攻击荷载；
4：同时显示 HTTP 请求；
5：同时显示 HTTP 响应头；
6：同时显示 HTTP 响应体。
