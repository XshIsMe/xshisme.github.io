information_schema 数据库

存储了数据库的信息

  • tables 表

    • table_schema 字段：数据库名
    • table_name 字段：表名

  • columns 表

    • table_schema 字段：数据库名
    • table_name 字段：表名
    • column_name 字段：字段名

  • database()：将会返回当前所使用的数据库名字
  • user()：将会返回执行当前查询的用户名
  • version()：获取当前数据库版本

常用函数

conv(N, from_base, to_base)

CONV() 函数的目的是在不同的数值基数之间进行数字转换。该函数返回值 N 从 from_base 到 to_base 转换的字符串。 最小基数为2，最大值为36。如果任何参数为 NULL，则函数返回 NULL

substr(string string,num start,num length)

string：字符串
start：起始位置（从1开始）
length：长度

limit 子句

i 为查询结果的索引值（默认从0开始），当 i=0 时可省略 i
n 为查询结果返回的数量
i 与 n 之间使用英文逗号 , 隔开
limit n 等同于 limit 0, n
