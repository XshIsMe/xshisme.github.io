在SQL注入无回显的情况下，可以使用延时注入，通过服务响应时间，判断是否成功注入

用到的函数

  • sleep(N)：让语句停留 N 秒钟，再继续执行
  • benchmark(count, expression)：重复计算 expression 表达式 count 次
  • if(condition, value_true, value_false)：判断 condition，结果为 true 返回 value_true，为 false 返回 value_false
  • ascii(char)：返回 char 的 ascii 码
  • left(string, N)：返回字符串 string 左边的 N 个字符
  • right(string, N)：返回字符串 string 右边的 N 个字符
  • substringing(string, start, length)：返回 string 字符串从第 start 个字符开始的 length 个字符
  • mid(string, start, length)：和 substringing() 一样
