Oah's Blog
2020-06-02
VSCode

vscode-icons

图标主题

One Monokai Theme

代码主题

Markdown PDF

MarkDown转PDF

  • 右键 → Markdown PDF: Export (pdf)

Markdown TOC

Markdown插入目录

  • 右键 → Markdown TOC: Insert/Update
  • Ctrl + M T

Python代码格式化

  • 文件 → 设置 → 首选项
  • 搜索 Python › Formatting: Provider
  • 选择black
  • Alt + Shift + F

code-python-isort

排序Python包导入

  • 右键 → 排序import语句
  • Alt + Shift + O

Python Docstring Generator

生成Python文档注释

  • 输入 ''' 然后按回车

Bracket Pair Colorizer

用不同颜色区分不同层次的括号

快速注释

  • Ctrl + /

一键生成html代码

  • 输入 !
  • 按回车

选择相同的字符

  • Ctrl + D
