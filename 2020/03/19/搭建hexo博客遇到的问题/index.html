Hexo

hexo

初始化hexo

在windows下如果用cmd执行 hexo init 会报错，要用git bash执行

安装上传到github的插件

npm install hexo-deployer-git --save 也要在git bash执行，如果在cmd执行，hexo d 的时候会报错 ERROR Deployer not found: git

在文章中插入图片

在站点配置文件中找到 post_asset_folder: false，改为 true，再执行 hexo n 的时候会在 _post 新建一个markdown文件和同名文件夹，将要用到的图片放进同名文件夹，在markdown文件中用 ![图片备注](图片名.jpg) 的方式引用
