v2rayи®ҫзҪ®

  • йҰ–е…Ҳи®ҫзҪ®v2rayе…Ғи®ёжқҘиҮӘеұҖеҹҹзҪ‘зҡ„иҝһжҺҘ
  • дё»з•Ңйқў вҶ’ еҸӮж•°и®ҫзҪ® вҶ’ v2rayNи®ҫзҪ® вҶ’ еӢҫйҖү вҖңе…Ғи®ёжқҘиҮӘеұҖеҹҹзҪ‘зҡ„иҝһжҺҘвҖқ

VMwareи®ҫзҪ®

  • иҰҒе°ҶзҪ‘з»ңжЁЎејҸж”№дёәжЎҘжҺҘжЁЎејҸ
  • йҖүдёӯиҷҡжӢҹжңәеҸій”® вҶ’ и®ҫзҪ® вҶ’ зҪ‘з»ңйҖӮй…ҚеҷЁ вҶ’ еӢҫйҖү вҖңжЎҘжҺҘжЁЎејҸпјҲBпјүпјҡзӣҙжҺҘиҝһжҺҘзү©зҗҶзҪ‘з»ңвҖқ

Linuxзі»з»ҹи®ҫзҪ®

жҹҘзңӢе®ҝдё»жңәзҡ„IPең°еқҖ

  • еңЁ cmd дёӯиҫ“е…Ҙ ipconfig

  • еҸҜд»ҘзңӢеҲ°жңүдёүдёӘйҖӮй…ҚеҷЁпјҢе…¶дёӯ VMnet1 жҳҜдё»жңәжЁЎејҸзҡ„зҪ‘еҚЎпјҢVMnet8 жҳҜNATжЁЎејҸзҡ„зҪ‘еҚЎпјҢWLAN жҳҜе®ҝдё»жңәWIFIзҡ„зҪ‘еҚЎ
  • жҲ‘д»¬йңҖиҰҒзҡ„жҳҜ WLAN зҡ„IPең°еқҖ

и®ҫзҪ®зі»з»ҹд»ЈзҗҶ

  • жүҫеҲ°Settings вҶ’ Network вҶ’ Network Proxy
  • еӢҫйҖү Manual
  • еңЁ Socks Host еӨ„еЎ«е…Ҙ WLAN зҡ„IPең°еқҖе’Ңv2rayзӣ‘еҗ¬зҡ„з«ҜеҸЈ

и®ҫзҪ®з»Ҳз«Ҝд»ЈзҗҶ

  • иҷҪз„¶и®ҫзҪ®дәҶзі»з»ҹд»ЈзҗҶпјҢдҪҶжҳҜз»Ҳз«ҜжҳҜдёҚдјҡиҮӘеҠЁиө°Socksд»ЈзҗҶзҡ„
  • еҸҜд»Ҙе®үиЈ…proxychainsжқҘиө°д»ЈзҗҶ
