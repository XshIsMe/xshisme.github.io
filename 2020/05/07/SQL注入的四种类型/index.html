floor жҠҘй”ҷжіЁе…Ҙ

з”ЁеҲ°зҡ„еҮҪж•°

  • concat(str1, str2, ...)пјҡе°ҶеӨҡдёӘеӯ—з¬ҰдёІиҝһжҺҘжҲҗдёҖдёӘеӯ—з¬ҰдёІ
  • floor()пјҡеҗ‘дёӢеҸ–ж•ҙпјҢеҸӘдҝқз•ҷж•ҙж•°йғЁеҲҶ
  • rand()пјҡз”ҹжҲҗ 0 еҲ° 1 д№Ӣй—ҙзҡ„йҡҸжңәж•°пјҢrand(0) еҸҜд»Ҙи®ҫзҪ®йҡҸжңәж•°з§ҚеӯҗпјҢжҜҸж¬Ўз»“жһңзӣёеҗҢ
  • version()пјҡиҝ”еӣһж•°жҚ®еә“зүҲжң¬

SQLиҜӯеҸҘ

жөӢиҜ•з»“жһң

еҺҹзҗҶеҲҶжһҗ

  • floor(rand()*2) йҡҸжңәз”ҹжҲҗ 0 жҲ– 1
  • group by иҝӣиЎҢеҲҶз»„ж—¶пјҢfloor(rand()*2) жү§иЎҢдёҖж¬ЎпјҲжҹҘзңӢеҲҶз»„жҳҜеҗҰеӯҳеңЁпјүпјҢеҰӮжһңиҷҡжӢҹиЎЁдёӯдёҚеӯҳеңЁиҜҘеҲҶз»„пјҢе°ұиҰҒжҠҠе®ғжҸ’е…ҘеҲ°иҷҡжӢҹиЎЁдёӯгҖӮйӮЈд№ҲеңЁжҸ’е…Ҙж–°еҲҶз»„зҡ„ж—¶еҖҷпјҢfloor(rand()*2) е°ұеҸҲи®Ўз®—дәҶдёҖж¬ЎгҖӮз¬¬дәҢж¬Ўиҝҗз®—зҡ„з»“жһңеҸҜиғҪдёҺз¬¬дёҖж¬Ўиҝҗз®—зҡ„з»“жһңдёҚдёҖиҮҙпјҢдҪҶжҳҜиҝҷдёӘиҝҗз®—зҡ„з»“жһңеҸҜиғҪеңЁиҷҡжӢҹиЎЁдёӯе·Із»ҸеӯҳеңЁдәҶпјҢйӮЈд№Ҳиҝҷж—¶зҡ„жҸ’е…Ҙеҝ…з„¶еҜјиҮҙдё»й”®зҡ„йҮҚеӨҚпјҢиҝӣиҖҢеј•еҸ‘й”ҷиҜҜ
  • еҪ“ group by еҜ№е…¶иҝӣиЎҢеҲҶз»„зҡ„ж—¶еҖҷпјҢйҰ–е…ҲйҒҮеҲ°з¬¬дёҖдёӘеҖј 0пјҢеҸ‘зҺ° 0 дёҚеӯҳеңЁпјҢдәҺжҳҜйңҖиҰҒжҸ’е…ҘеҲҶз»„пјҢе°ұеңЁиҝҷж—¶пјҢfloor(rand()*2) еҶҚж¬Ўиў«и§ҰеҸ‘пјҢз”ҹжҲҗз¬¬дәҢдёӘеҖј 1пјҢеӣ жӯӨжңҖз»ҲжҸ’е…ҘиҷҡжӢҹиЎЁзҡ„д№ҹе°ұжҳҜз¬¬дәҢдёӘеҖј 1пјӣз„¶еҗҺйҒҮеҲ°з¬¬дёүдёӘеҖј 1пјҢеӣ дёәе·Із»ҸеӯҳеңЁеҲҶз»„ 1 дәҶпјҢе°ұзӣҙжҺҘи®Ўж•°еҠ  1пјҲиҝҷж—¶ 1 зҡ„и®Ўж•°еҸҳдёә 2пјүпјӣйҒҮеҲ°з¬¬еӣӣдёӘеҖј 0 зҡ„ж—¶еҖҷпјҢеҸ‘зҺ° 0 дёҚеӯҳеңЁпјҢдәҺжҳҜеҸҲйңҖиҰҒжҸ’е…Ҙж–°еҲҶз»„пјҢз„¶еҗҺ floor(rand()*2)еҸҲиў«и§ҰеҸ‘пјҢз”ҹжҲҗз¬¬дә”дёӘеҖј 1пјҢеӣ жӯӨиҝҷж—¶иҝҳжҳҜеҫҖиҷҡжӢҹиЎЁйҮҢжҸ’е…ҘеҲҶз»„ 1пјҢдҪҶжҳҜеҲҶз»„ 1 е·Із»ҸеӯҳеңЁдәҶпјҢжүҖд»ҘжҠҘй”ҷ
