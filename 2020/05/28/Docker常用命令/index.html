Docker常用命令

Oah's Blog
2020-05-28
开发
Docker

镜像

镜像命名方式

列出本地主机上的镜像

  • -q：只显示镜像ID

删除镜像

  • -f：强制删除

根据Dockerfile创建镜像

  • -t：设置镜像名
  • -f：指定Dockerfile路径

容器

新建并启动容器

  • –name：为容器指定一个名称
  • -d：后台运行容器，并返回容器 ID，也即启动守护式容器
  • -it：前台运行容器
  • -p：指定端口映射，-p 容器外端口:容器内端口

列出当前所有正在运行的容器

  • -a：列出当前所有正在运行 + 历史上运行过的容器
  • -q：只显示容器编号

容器内退出容器

  • 停止容器并退出
  • 不停止容器退出

进入容器

启动停止的容器

重启容器

停止容器

  • 相当于电脑正常关机

强制关闭容器

  • 相当于电脑拔电源关机

向容器传入命令

  • 在容器中打开新的终端，并且可以启动新的进程（没有进容器，直接得到结果）

查看容器

  • -a：显示所有（包括已停止的）容易
  • -q：只显示容器ID

删除容器

  • -f：强制删除

Dockerfile

FROM

  • 基础镜像，当前镜像是基于哪一个镜像的

RUN

  • 容器构建时需要运行的命令

WORKDIR

  • 指定在创建容器后，终端默认登录进来的工作目录，一个落脚点

ADD

  • 拷贝 + 解压缩
  • 将宿主机目录下的文件拷贝进镜像且ADD命令会自动处理url和解压tar压缩包

CMD

  • 指定一个容器启动时要运行的命令
